# DEFERRED DECISIONS

> Решения, отложенные командой «не знаю» / «пропусти» / «позже» во время интервью или разработки.
> Агент проверяет этот файл в начале каждой сессии и напоминает об открытых вопросах.

## Правила

- Не удалять записи — только менять статус на ✅ Решено
- Агент напоминает: «У тебя N открытых решений — разберём?"
- При принятии решения — обновить соответствующий документ

## Шаблон

```
### [Тема решения]
- Документ: [ARCHITECTURE.md / ROADMAP.md / и т.д.]
- Контекст: [почему отложили]
- Варианты: [если обсуждали]
- Статус: ⏸️ Открыто / ✅ Решено
- Дата: YYYY-MM-DD
```
