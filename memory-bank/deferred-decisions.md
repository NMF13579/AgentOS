# DEFERRED DECISIONS

> Отложенные решения. Агент добавляет запись при команде «не знаю», «пропусти», «позже».

## Инструкция для агента

- При ответе «не знаю» / «пропусти» / «позже» — добавить запись сюда
- В начале следующей сессии — показать список открытых решений
- После принятия решения — пометить статус «✅ Принято» и обновить нужный документ

## Шаблон

```
### [Тема решения]
- Документ: [файл, который нужно обновить]
- Контекст: [почему нужно это решение]
- Варианты: [если агент предлагал — перечислить]
- Статус: ⏸️ Отложено / ✅ Принято / ❌ Отменено
- Дата: YYYY-MM-DD
```
