# DEFERRED DECISIONS

> Решения, отложенные через механизм DEFER. Агент напоминает об открытых вопросах в начале каждой сессии.

## Инструкция для агента

При старте сессии:
1. Прочитать этот файл
2. Если есть открытые пункты — сообщить: «У тебя N отложенных решений — разберём?»
3. Разбирать по одному, не всё сразу
4. После принятия решения — обновить документ и соответствующий файл

## Шаблон записи

```
### [Тема решения]
- Файл: docs/ARCHITECTURE.md (раздел: Стек)
- Контекст: [почему отложили]
- Варианты: 1. [...] 2. [...] 3. [...]
- Срочность: высокая / средняя / низкая
- Статус: ⏸️ открыто / ✅ решено
- Решение: [заполнить после]
```
