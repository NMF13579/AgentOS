# QUICK START — запуск без терминала

> Первая настройка через интерфейс Cursor/Claude Code.

Перед стартом пройди мастер для новичков:  
[`ONBOARDING-WIZARD.md`](./ONBOARDING-WIZARD.md)

---

## Шаг 1️⃣ — Открой проект в IDE

- Открой папку проекта в Cursor или Claude Code.
- Убедись, что в корне видны `START.md`, `README.md`, `LAYER-1/`, `LAYER-2/`, `LAYER-3/`.

---

## Шаг 2️⃣ — Запусти агент из интерфейса

- Нажми `Cmd/Ctrl + K`.
- Вставь шаблон запроса:

```md
Прочитай `START.md`, затем `llms.txt`, затем `HANDOFF.md` и `LAYER-3/project-status.md`.
Коротко скажи, где мы находимся, предложи 1 следующий шаг и жди подтверждения.
```

- Нажми **Run**.

---

## Шаг 3️⃣ — Подтверди следующий шаг

- Агент покажет текущее состояние и предложит безопасный следующий шаг.
- Подтверди действие коротко: `да`, `выполняй` или `go`.
- После выполнения попроси обновить `HANDOFF.md` и `LAYER-3/project-status.md`.
