# AgentOS Release Notes

## 0.2.0-rc.1 — Public MVP Candidate

### Status
This is a **public MVP candidate / release candidate**. It is intended for evaluation and feedback in non-critical environments. It is **not** a production release and does not declare final public release completion.

### What This Candidate Includes
- **Single-Role Execution Guard:** Prevents role mixing during agent runs.
- **Unified Validation Suite:** One-command check for repository health.
- **Task Contract System:** Structured YAML-first governance for AI changes.
- **Pilot Hardening:** Infrastructure for collecting and classifying external feedback.
- **Public Limitations:** Transparent disclosure of current boundaries.
- **Troubleshooting Base:** Initial scenarios for common usability issues.

### What This Candidate Does Not Claim
- **Production Readiness:** Not suitable for production deployment or scale.
- **Production-Grade Sandboxing:** Does not provide secure execution isolation.
- **Guaranteed Safe AI Output:** AI changes still require rigorous manual review.
- **Bug-Free Result:** AgentOS is an experimental framework.
- **Automatic Approval:** Human gates are mandatory and non-bypassable.
- **SaaS Readiness:** Currently local/CLI only.
