# Known Limitations

## Purpose

This document records known non-blocking limitations of AgentOS during the M38 pilot hardening phase.

Known limitations are not resolved issues and are not approvals to bypass safety checks.

## Rules

- P0/P1 pilot blockers must not be hidden here.
- Known limitations must not weaken safety boundaries.
- Known limitations must not authorize unsupported workflows.
- Known limitations must not claim public release readiness.
- Each limitation must include impact and workaround if known.

## Limitation Entry Format

### LIM-001 — Example Title

- Source feedback ID: M38-FB-EXAMPLE
- Severity: P2 / P3 / UNKNOWN_NON_BLOCKING
- Type: Documentation Gap
- Impact: User may require additional context for advanced setup.
- Workaround: Refer to internal technical architecture docs.
- Pilot blocking: NO
- Planned handling: Polish documentation in Milestone 39.
- Related report: reports/m38-pilot-feedback-intake.md
