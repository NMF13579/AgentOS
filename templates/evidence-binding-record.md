# Evidence Binding Record Template

Purpose: bind PASS claim to artifact hashes with deterministic binding hash.

Evidence Binding is integrity evidence, not approval.
PASS claim with agent-computed binding only is not clean PASS.

Use deterministic input:
<path>|<sha256>

One line per artifact, sorted by path, joined with \n.

Then compute:
sha256:<sha256 of joined text>

Clean PASS requires:
- artifacts exist
- hashes match
- binding hash reproducible
- generated by trusted evaluator/runner/wrapper

```json
{
  "result_claim": "PASS",
  "evidence_binding": {
    "bound_artifacts": [
      {
        "path": "reports/ci/agentos-validate.json",
        "sha256": "sha256:..."
      }
    ],
    "binding_hash": "sha256:...",
    "generated_by": "trusted-evaluator",
    "validation_source_id": "agentos-runner"
  }
}
```
