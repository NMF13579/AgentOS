# LESSONS LEARNED — накопленные уроки проекта

> Здесь хранятся не только фиксы, но и повторно полезные выводы.
> Если правило может предотвратить будущую ошибку — оно должно быть записано здесь.

## Когда обновлять

Урок записывается если выполнено хотя бы одно условие:

- Задача потребовала RETRY RULE
- Была ошибка типа PROCESS / LOGIC / SCOPE / CONTEXT ERROR
- Решение заняло дольше ожидаемого
- Владелец сказал «запомни» / «больше так не делай»
- Одна и та же проблема встретилась второй раз
